    Struct SpanOwner<T>

    Represents the memory obtained from the pool or allocated on the stack or heap.

    T

    The type of the elements in the rented memory.
    Remarks

    This type is aimed to be compatible with memory allocated using stackalloc operator. If stack allocation threshold is reached (e.g. StackallocThreshold) then it's possible to use pooled memory from arbitrary MemoryPool<T> or Shared. Custom ArrayPool<T> is not supported because default Shared is optimized for per-CPU core allocation which is perfect when the same thread is responsible for renting and releasing the array.

    Examples
    const int stackallocThreshold = 20;
var memory = size <=stackallocThreshold ? new SpanOwner<byte>(stackalloc byte[stackallocThreshold], size) : new SpanOwner<byte>(size);
