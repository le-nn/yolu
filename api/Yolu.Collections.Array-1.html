    As<TOther>()

    Creates an immutable array for this array, cast to a different element type.

    Declaration
    [Pure]
public Array<TOther> As<TOther>() where TOther : class
    Returns
    Type Description
    Array<TOther>

    A struct typed for the base element type. If the cast fails, an instance is returned whose IsDefault property returns true.
    Type Parameters
    Name Description
    TOther

    The type of array element to return.
    Remarks

    Arrays of derived elements types can be cast to arrays of base element types without reallocating the array. These upcasts can be reversed via this same method, casting an array of base element types to their derived types. However, downcasting is only successful when it reverses a prior upcasting operation.
