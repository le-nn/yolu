    Class NumberExtensions

    Extension methods on int to allow for a more fluent way of specifying a TimeSpan.

    Inheritance
    object
    NumberExtensions
    Inherited Members

    Namespace: Yolu.DateTimes
    Assembly: Yolu.dll
    Syntax
    public static class NumberExtensions
    Examples

    Instead of

    TimeSpan.FromHours(12)

    you can write

    12.Hours()

    Or even

    12.Hours().And(30.Minutes()).
